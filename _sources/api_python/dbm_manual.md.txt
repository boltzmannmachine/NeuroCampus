# `neurocampus.models.dbm_manual`

Módulo dedicado a la implementación manual de Deep Boltzmann Machines para
experimentos y pruebas dentro del proyecto.

## Elemento principal

- `DBMManual`: implementación de referencia para entrenamientos DBM manuales.

## Uso esperado

Se emplea desde jobs de entrenamiento y desde flujos de experimentación donde
se necesita controlar directamente la configuración del modelo.

## Relación con otras secciones

- La justificación teórica se describe en [Deep BM](../teoria/deep_bm).
- La integración del pipeline se resume en [Pipeline de modelos](../arquitectura/pipeline_modelos).
