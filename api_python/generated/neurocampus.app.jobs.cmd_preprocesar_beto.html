neurocampus.app.jobs.cmd_preprocesar_beto

Job CLI para: - ejecutar BETO (sentiment) sobre texto lematizado - opcionalmente generar embeddings TF-IDF + LSA (64 dims) -> feat_t_1..feat_t_64 - persistir dataset etiquetado y meta.json (auditoría)

Cambios clave (DataTab / Datos)

  1. Añade –empty-text-policy para controlar comentarios vacíos:

    • neutral (legacy): texto vacío se etiqueta como neutral (p_neu=1, accepted=1)

    • zero (recomendado): texto vacío se etiqueta como NO_TEXT (p_neg=p_neu=p_pos=0, accepted=0, label=no_text)

  2. Mantiene compatibilidad con –keep-empty-text: - si –empty-text-policy no se envía:

    • neutral si keep_empty_text=True

    • zero si keep_empty_text=False

  3. Si se generan embeddings TF-IDF+LSA, siempre se fuerza vector 0 en filas sin texto.
